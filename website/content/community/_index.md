Mara is **Apache 2.0** open source. We welcome issues and pull requests from individuals and organizations.

## Contributing

- Read [**CONTRIBUTING.md**](https://github.com/ArdurAI/mara/blob/main/CONTRIBUTING.md) on GitHub for the full workflow (fork, branch, DCO sign-off, `cargo fmt` / `clippy` / `test`).
- Non-trivial changes should start with an **issue** so maintainers can help scope the work.
- Architecture decisions belong in [`docs/adr/`](https://github.com/ArdurAI/mara/tree/main/docs/adr/).

## Code of conduct

We follow the [Contributor Covenant](https://github.com/ArdurAI/mara/blob/main/CODE_OF_CONDUCT.md). Be respectful, assume good intent, and help keep discussions technical and inclusive.

## Security

**Do not** open public issues for unfixed vulnerabilities. Email **security@ardurai.dev** — see [**SECURITY.md**](https://github.com/ArdurAI/mara/blob/main/SECURITY.md) for scope, disclosure timeline, and advisory process.
